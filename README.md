# Python-Dodgegame

## About

A simple dodge game by pygame, for studying good practice of python programming and practicing distributing python project.

## How to play

### From source

1. Clone the repo ```git clone https://github.com/pupungeggang4/Python-Dodgegame.git```

2. Go to src/ folder.

3. Run python3 main.py.

### From release (Windows)

1. Go to release file.

2. Download file.

3. Launch exe.
